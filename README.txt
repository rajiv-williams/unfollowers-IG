Install the following packages:

    -instaloader
    -itertools
    -threading
    -time
    -sys

Use Git Bash to run the program on Windows by using the following command:

    bash ./unfollowers.sh

The program is unstable and can give errors if there are too many frequent 
requests, or if login info is incorrect. 

WARNING: If there are too many frequent requests, Instagram may block your
IP temporarily, or lock your account for security measures. 
The owner of this GitHub repository is NOT responsible for anything that 
happens to your Instagram account.
